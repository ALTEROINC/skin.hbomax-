# Inter Font Setup

Download Inter from:
`https://fonts.google.com/specimen/Inter`

Place these exact font files in the `fonts/` directory:

- `Inter-Regular.ttf`
- `Inter-Medium.ttf`
- `Inter-SemiBold.ttf`
- `Inter-Bold.ttf`

These font files are not included in the repo due to font licensing and must be added manually before the skin will load.
